# ES-PI1-2026-T103-G06 - Projeto LAD.Py
Repositório do Sistema de Votação digital fictício LAD.Py; Turma 103 e Grupo 06 - Projeto Integrador I, Engenharia de Software; Instituição PUC Campinas

Este Projeto 

### Descrição do Projeto

O projeto LAD.PY consiste no desenvolvimento do backend de um sistema de votação digital fictício, com finalidade estritamente didática, cujo objetivo é simular o funcionamento básico de um programa de votação, além da organização, registro e segurança dos dados.

A proposta busca integrar, de forma prática, conhecimentos de programação em Python, manipulação de bancos de dados com SQL e aplicação de conceitos matemáticos, como Álgebra Linear, com foco na proteção da informação.
Este foi projetado epara garantir a identificação dos participantes, o registro dos votos e a verificação dos resultados, aplicando validação de dados, integridade, uso de logs e criptografia para proteção das informações. A aplicação é executada via terminal, e o banco de dados é aberto localmente.
